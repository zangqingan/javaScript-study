1、JS语法基础相关概念(词法结构)

任何语言的核心所描述的都是这门语言是如何在最基本的层面上如何工作的,涉及语法、操作符、数据类型以及内置功能。 而JavaScript这门语言的核心特性实现最广泛的版本是以ECMAScript的伪语言形式定义ECMA-262规范的版本我们称之为ES5。

1.基本语法:

JavaScript的语法借鉴了C语言和其它类C语言如Java语言、Perl,区别是js是动态的解释性语言(即代码是从上往下一行一行读取然后执行,变量的类型是动态的即可以随时改变)所以语法是比较宽松的。
1.js严格区分大小写
ECMAScript中一切都区分大小写,所以text和Text是两个不同的变量,如果弄错了大小写程序将报错或者运行不正常, 所以在给变量命名,函数命名和使用操作符关键字时要注意区分大小写。操作符、关键字都是小写的,而变量/函数名等命名遵循小驼峰命名法。 另外js会忽略多余的空格,制表符和换行符(这三者统称为分隔符)。空格的作用就是用来分隔字符序列。
2.js注释
js注释是用于对代码调试或说明情况可以提高代码的的可读性它是给开发人员看的,JavaScript引擎会自动忽略(即不解析)。 ecmascript使用c语言风格的注释包括单行注释和块级注释,还有一种是函数形参注释说明。 //单行注释,以//开头直到行末的字符被视为单行注释, /*多行块级注释*/,用/*...*/把多行字符包裹起来,把一大“块”视为一个注释。 但是块注释对于被注释的代码块存在安全隐患,比如代码块中存在正则字面量表达式时。 所以要避免使用块注释,统一使用单行注释。 /** *给函数的文档注释,在调用函数时会显示这个函数的信息。 */ 注意:注释一定要精确的描述代码不然没用的注释比没注释还糟糕。
3.js字符集
JavaScript 使用的是32位的Unicode 字符集(也就是基本地球上啥语言都包含的字符集),它是一个构成js程序字符的集合。
4.js标识符
标识符(identifier)指的是用来识别各种值的合法名称。 也就是在js中能自己起名的,如变量名,属性名,函数名或函数参数名等,它的组成规则如下: 第一个字符,可以是任意 Unicode 字母(包括英文字母和其他语言的字母),以及美元符号($)和下划线(_)。 第二个字符及后面的字符,除了 Unicode 字母、美元符号和下划线,还可以用数字0-9。 注意:标识符不能使用js语言的关键字、保留字。 简单说就是不能以数字开头就行,然后一般是英文字母开头后面可选择地加上一个或者多个字母、数字、下划线。 尽量使用小驼峰命名法(第一个单词首字母小写,后面的每一个单词首字母大写)或-短横线分隔命名法。 tips:使用中文本质上也是可以的不过不推荐这样避免产生不必要的错误,同时尽量使用语义化命名即可。
5.js严格模式
ECMAScript5增加了严格模式(strict mode)也叫现代模式,它是一种不同的 JavaScript解析和执行模型,一些不规范的写法在这个模式下会被处理。 启用严格模式:在脚本文件开头加上 "use strict"; 只能放到第一行,则整个脚本文件才能以“现代”模式进行工作。 这样ES5 规范中新增加并且修改了的一些已经存在的特性将会启用。
6.js语句
js语句(statement)是为了完成某种任务用于向浏览器发送命令而进行的操作代码,js中语句结束以分号标识(;可选) js语句也分成很多种常见的有声明语句、赋值语句、条件判断语句、循环语句等等,它们都有自己特定的作用。 js语句通常按照从上到下、从左到右的顺序执行。
7.js运算符(操作符)
js运算符是语言本身提供的用来对js中的数据(变量)进行操作(赋值,加减乘除求余,逻辑运算等等操作)的符号。 JavaScript 提供了多种运算符,覆盖了所有主要的运算。
8.js表达式
js常量、js变量和js运算符组成的js语句就称为表达式。
9.js代码块
多条语句时可以放到一个C语言风格的代码块中,代码块由一对花括号标识。 { 语句1; 语句2; 语句...; 语句n; } 代码块没有什么特殊含义,它作用就是一起执行花括号中的内容,可以让内容更清晰。
10.js关键字和保留字:
ECMA-262规范中描述了一组保留的关键字和一组未来的保留字,它们都有特殊的用途即作为这门语言本身的一部分。 所以都不能用作标识符或属性名 关键字 js语言中本身自带的(js这门语言已经使用的)已经使用的标识符用于标识要执行的操作,如var用于声明变量。 保留字 将来可能用作关键字的字(其实就是一些英语单词)。
11.js字面量
直接量(也叫字面量literal)即程序中直接显示出来的数据值一个恒定的值(不可改变)。 字面量都可以直接使用但是一般不直接使用而是使用一个变量名代替。 就好比我们用人名对应某个人一样。 常见字面量种类如下所示 100 3.14 123e3 //数字字面量,可以是整数或者是小数,或者是科学计数(e) '王耿' "wangg" //字符串字面量,用单引号或者双引号括起来的。 true/false //布尔值字面量 x+y //表达式字面量用于计算 /JavaScript/gi //正则表达式字面量 null //空对象字面量 {x:1,y:2} //对象字面量表达式定义了一个对象 [1,2,3] //数组字面量表达式定义了一个数组 function functionName(形参){return语句} //函数字面量表达式定义了一个函数

2.js变量

ECMASccript变量是送散类型的,即它可以用来保存任意js数据类型的数据。变量就是对“值”的具名引用。变量就是为“值”起一个别名,然后引用这个名字,就等同于引用这个值。 变量的概念基本上和初中代数的方程变量是一致的,只是在计算机程序中,变量不仅可以是数字,还可以是任意数据类型。 变量的名字就是变量名,它遵守js标识符的命名规范。注意 变量名也可以用中文,但是,请不要给自己找麻烦。 在JavaScript中有3个关键字可以用来声明变量:var、let、const,其中var是所有版本都可以使用而let和const只能在es6及更晚的版本中使用。

3.js数据类型

计算机顾名思义就是可以做数学计算的机器,因此,计算机程序理所当然地可以处理各种类型的数值。 也就是说计算机能处理的远不止数值,还可以处理文本、图形、音频、视频、网页等各种各样的数据。 为了让计算机能够识别我们传入的数据是哪种类型的,就需要我们提前在语言的底层声明不同的数据对应的数据类型是什么。 目前在ECMAScript中定义了8种数据类型,其中Undefined、Null、Boolean、Number、BigInt、String、Symbol七种为简单数据类型(也叫原始类型primitive type), 其中BigInt、Symbol是ECMAScript6之后新增的。还有一种复杂数据类型(也叫合成类型complex type)叫对象Object,它是一种无序名值对的集合。 JavaScript 语言中的每一个值,都必定属于这八种中的某一种数据类型。 undefined 表示“未定义”或不存在,即此处暂时没有任何值。 null 表示空值,即此处的值为空。 boolean 布尔类型-表示真伪的两个特殊值,即true(真)和false(假)。 number 数值类型-包括整数和小数(比如1和3.14)。 bigint 数字，用于表示任意长度的整数。 string 字符串类型-也就是我们常说的文本 "Hello World" , 'hi' 必须使用双/单引号包裹起来。 symbol 符号类型-表示独一无二的值 object 对象类型-各种值组成的集合。
