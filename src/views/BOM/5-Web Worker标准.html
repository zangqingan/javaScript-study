1.概述

            我们知道JavaScript语言采用的是单线程模型、而为了利用多核 CPU 的计算能力 HTML5 提出 Web Worker 标准。
            Web Worker 的作用就是为 JavaScript 创造多线程环境、允许主线程创建 Worker 线程、将一些任务分配给后者运行。
            Worker 线程一旦新建成功就会始终运行不会被主线程上的活动打断、这样有利于随时响应主线程的通信。
            缺点是比较耗费资源、所以一旦使用完毕就应该关闭。
            浏览器原生提供Worker()构造函数，用来供主线程生成 Worker 线程。
