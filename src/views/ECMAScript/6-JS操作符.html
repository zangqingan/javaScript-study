1、js操作符:

1.js操作符简介:

js操作符也叫操作符是ECMAScript-262中定义的用来操作数据值的一组符号(本质是执行特定功能的符号)。 通过操作符可以对一个或多个数值(直接量或者变量)进行某种操作,并获取操作后的结果。

2.js常见操作符及分类:

1.算术操作符

加+ 减- 乘* 除/ 求余%,对非数值运算时先转为数值再运算。(除了字符串的加法,它是拼串操作。) 任何值做-,*,/运算时都会自动转换为number可以利用这一特点做隐式类型转换,即通过-0,*1,/1就可以实现,原理和Number()函数一样。
ES2016 新增了一个指数运算符 ** 3 ** 2 ==> 4,之前是使用Math.pow(3,2)

2.一元操作符

只操作一个操作数的操作符,+,-。正负号。自增++,自减--可以在自身的基础上增加或减少1。 注意:a和a++,++a是不一样的,a是变量,a++和++a是表达式。 无论使用前置还是后置的自增或自减,变量a都会增加或减少1。 不同在于a++和++a是表达式的值不同 a++ 等于原变量的值, ++a 等于原变量新值。

3.逻辑操作符

!非,&&与(且),||或。将多个操作数放在一起运算返回结果是布尔值。
and与&&,有假且假,短路与,所以左操作数为false时直接返回false,不再判断右边的表达式。 or或||,有真或为真,短路或,所以左操作数为true时直接返回true,不再判断右边的表达式。 not非!,取反。 ES2020 引入了一个新的 Null 判断运算符??。它的行为类似||,但是只有运算符左侧的值为null或undefined时,才会返回右侧的值。 ES2021 引入了三个新的逻辑赋值运算符logical assignment operators,将逻辑运算符与赋值运算符进行结合。 ||=、&&=、??=相当于先进行逻辑运算,然后根据运算结果,再视情况进行赋值运算。 // 老的写法 user.id = user.id || 1; // 新的写法 user.id ||= 1;

4.赋值操作符

= 可以将符号右侧的值赋值给符号左侧的变量,可以和算术操作符组合成复合赋值使用 如: a+=5 是 a=a+5的缩写。 复合赋值只是简写语法,不会提升性能。


5.关系操作符

用来比较两个变量(值)之间的大小关系,成立返回true,不成立返回false。 大于>,大于等于>=,小于<,小于等于<=,等于==,不等于!=,全等于===,不全等于!==。 注意:返回结果是布尔类型的值true/false 等于 在比较时会进行类型的隐式转换。 全等则必须值和类型都一样才会返回true, 面试题:何时使用"===",何时使用 "==" "==" 只做值的比较且在比较时会进行类型的隐式转换 "===" 比较值和类型

6.三元操作符

语法:条件?语句1:语句2;条件成立时执行语句1,不成立执行语句2。

7.逗号操作符

用来在一条语句中执行多个操作,let n = 1,m=3,p=4; 主要用来一条语句中同时声明多个变量。

3.js操作符的优先级

JavaScript 各种运算符的优先级别(Operator Precedence)是不一样的它能决定操作符执行的先后顺序。优先级高的运算符先执行,优先级低的运算符后执行。 圆括号（()）可以用来提高运算的优先级,因为它的优先级是最高的,即圆括号中的表达式会第一个运算。 注意:圆括号不是运算符,而是一种语法结构。它一共有两种用法:一种是把表达式放在圆括号之中,提升运算的优先级；另一种是跟在函数的后面,作用是调用函数。
